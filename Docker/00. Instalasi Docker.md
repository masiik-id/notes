# Docker
## Instalasi Docker di Linux
```bash
curl -fsSL https://get.docker.com -o get-docker.sh && sh get-docker.sh
```
## Penulisan Volume di WSL 
```bash
volumes:
    - /home/$USER/volumes/

Contoh : 
      - /home/$USER/volumes/sqlserver-data:/var/opt/mssql
```


# Docker Compose

### Build atau rebuild services
```bash
docker compose build
```

### Create & start all services (foreground)
```bash
docker compose up
```

### Start all services (background/detached mode)
```bash
docker compose up -d
```

### Stop and remove containers, networks, volumes, images
```bash
docker compose down
```

### Hanya menghentikan container (tanpa remove)
```bash
docker compose stop
```

### Menyalakan container yang sudah pernah dibuat
```bash
docker compose start
```

### Restart container
```bash
docker compose restart
```

### Melihat container yang berjalan
```bash
docker compose ps
```

### Melihat log service secara live
```bash
docker compose logs -f
```

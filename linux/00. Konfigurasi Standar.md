# Konfigurasi Standart

### o Update & Upgrade Sistem
```bash
sudo apt update && sudo apt upgrade -y
sudo apt autoremove -y
```

### o Install Paket Dasar & Penting
```bash
sudo apt install -y \
  build-essential \
  curl wget git unzip zip \
  software-properties-common \
  ca-certificates \
  net-tools \
  htop \
  ufw \
  gnupg lsb-release
```


### o Setup Firewall UFW
```bash
sudo ufw allow OpenSSH
sudo ufw enable
sudo ufw status
```

### o Install Git & Set Global Config
```bash
git config --global user.name "Nama Anda"
git config --global user.email "email@domain.com"
git config --global init.defaultBranch main
```

### o Install Python + Pip + Virtualenv
```bash
sudo apt install -y python3 python3-pip python3-venv
```

### o Install SSH Key (jika mau pakai git/remote server)
```bash
ssh-keygen -t ed25519 -C "email@domain.com"
cat ~/.ssh/id_ed25519.pub
```

### o Bersihkan sistem
```bash
sudo apt autoremove -y
sudo apt autoclean -y
```
